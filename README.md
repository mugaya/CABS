# CABS
Clinic Appointment Booking System (CABS) to manage patient bookings.

## Technology stack
Java - 25\
Openjfx - 21


## Features
### Admins
Register and log in\
Manage doctor availability - add and remove doctors


### Patients
Register and log in\
View all available appointment slots\
Book, reschedule or cancel appointments


### Limitations
No data persistence as there is no database implementation\
Credentials have been saved in plain text (No hashing)


## Installation

git clone https://github.com/mugaya/CABS.git


cd CABS\
